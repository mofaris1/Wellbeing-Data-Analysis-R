# Subjective Wellbeing & Mental Health Analysis in Jordan

## Overview
This project presents a comprehensive statistical analysis of subjective well-being, mental health, and food security in Jordan. Using R, the project processes survey data from over 1,000 participants to explore the intersection of lifestyle behaviors, stress mindsets, and socioeconomic factors, translating statistical findings into actionable policy recommendations.

## Datasets
- **Cleaned Wellbeing Dataset:** Contains responses covering the Food Insecurity Experience Scale (FIES), Resilience Scale (RS), Kessler Psychological Distress Scale (K10+), and Stress Mindset Measure (SMM).

## Key Objectives & Tasks
- **Data Wrangling & Cleaning:** Utilized `dplyr` and `tidyr` to clean, transform, and aggregate complex survey data.
- **Exploratory Data Analysis (EDA):** Conducted statistical summarization to understand distributions of stress and resilience across different demographics (e.g., gender, income, physical activity).
- **Data Visualization:** Built informative and aesthetic visualizations using `ggplot2` to illustrate trends, such as the correlation between household income and dietary habits.
- **Actionable Insights Generation:** Authored both a Technical Report (detailing statistical methodologies) and an Analytical Report (providing strategic interventions like "Hunger-Free Campus" and "Student Wellness Centers").

## Technologies Used
- **Language:** R
- **Libraries:** `ggplot2`, `dplyr`, `tidyr`, `reshape2`, `RMarkdown`
- **Reporting:** Statistical Analysis, Policy Recommendation Writing
